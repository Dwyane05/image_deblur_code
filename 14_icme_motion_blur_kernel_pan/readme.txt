This package contains an implementation of the image deblurring algorithm described in the paper: Jinshan Pan, Risheng Liu, Zhixun Su, and Guili Liu: Motion blur kernel estimation via salient edges and low rank prior, ICME 2014.

Please cite our paper if you use the code to generate results.


Acknowledgments:
Our implementation is based on the following papers:
(1) Jinshan Pan, Risheng Liu, Zhixun Su, and Xianfeng Gu, Kernel estimation from salient structure for robust motion deblurring, Signal Processing: Image Commuincation, vol. 28, no.9, pp. 1156¨C1170, 2013.
(2) Weisheng Dong, Guangming Shi, and Xin Li, Nonlocal image restoration with bilateral variance estimation: a low-rank approach, IEEE TIP, vol. 22, no. 2, pp. 700-711, 2013.
(3) Anat Levin, Yair Weiss, Fredo Durand, and William T. Freeman, Efficient marginal likelihood optimization in blind deconvolution, CVPR, 2011, pp. 2657¨C2664.
